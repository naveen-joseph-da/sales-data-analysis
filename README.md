# Sales Data Analysis

## Project Overview
This project focuses on analyzing sales data to identify trends, top-performing products, and monthly revenue patterns.

## Tools & Technologies
- Python
- Excel
- Pandas
- Matplotlib

## Key Analysis Performed
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Top-selling products analysis
- Revenue summary

## Objective
To gain insights from raw sales data and present meaningful business conclusions using data analysis techniques.
